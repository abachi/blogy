## Blogy

A simple blog built with [reactjs](https://reactjs.org) and firebase.

## Run locally
1. Clone this repo via terminal
```bash
  git clone git@github.com:abachi/blogy.git
```
2. Enter blogy and install dependencies
```bash
  cd blogy && npm install
```
3. Run local server
```bash
  npm start
```
4. Run tests
```bash 
  npm tests
```
